R.evolve

A transformation functions that transforms object recursively according to input transformation object 

          
const user = {
  favoriteRepos: ['ramda', 'react', 'nodejs'],
};
const transforms = {
  favoriteRepos: R.join(', '),
};
const result = R.evolve(transforms, user);

/*
{
  favoriteRepos: 'ramda, react, nodejs',
}
*/
